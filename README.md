nodejs构建简单电影网站<br/>
常见问题<br/>
npm 安装bootstrap报错path 解决：使用Git Bash来代替cmd 命令 bower install bootstrap<br/>
mongodb 安装启动<br/>
mongod --install --directoryperdb --serviceName MongoDB <br />
mongod --logpath "D:\mongodb\log\mongodb.log" --dbpath "D:\mongodb\db" <br/>
*(需在d盘创建对应的文件夹以及文件)
